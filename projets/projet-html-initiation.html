Résumé du projet

Premier projet de formation : découverte des fondamentaux du développement web en créant une page HTML statique simple. L'objectif est de comprendre la structure d'une page web, d'apprendre à utiliser les balises HTML sémantiques et d'appliquer des styles CSS de base.

  • Structure HTML : mise en place d'une structure de page claire avec header, main, footer et sections sémantiques.
  • Mise en forme CSS : application de styles basiques (couleurs, typographie, espacements) pour rendre la page attrayante.
  • Bonnes pratiques : respect de la sémantique HTML, code propre et indenté, validation W3C.

Détails clés

  • Client Formation OpenClassrooms
  • Rôle Développeur Junior
  • Durée 10h
  • Livrables Page web statique structurée avec HTML sémantique

Résultats finaux

Page web statique fonctionnelle démontrant la maîtrise des bases HTML/CSS. Structure sémantique claire, styles appliqués de manière cohérente et code validé selon les standards W3C.

  • HTML sémantique : utilisation appropriée des balises header, nav, main, section, footer.
  • CSS organisé : styles appliqués avec sélecteurs de classe, code bien indenté et commenté.
  • Standards respectés : validation W3C réussie, code conforme aux bonnes pratiques.
  • Base solide : fondations acquises pour aborder des projets plus complexes.
