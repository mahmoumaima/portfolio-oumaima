Le Contexte

Comprendre le challenge

Mission

Premier projet de formation OpenClassrooms : découverte des fondamentaux du développement web en créant une page HTML statique simple. L'objectif est de comprendre la structure d'une page web et d'apprendre les bases.

Objectif

Apprendre à utiliser les balises HTML sémantiques, structurer correctement une page web et appliquer des styles CSS de base pour rendre la page attrayante et lisible.

Livrables

  • Page HTML structurée
  • Feuille de styles CSS
  • Code validé W3C
  • Bonnes pratiques

Détails clés

  • Durée 15h

Les Résultats

Mission accomplie

Première page web créée avec succès : structure HTML5 sémantique maîtrisée, styles CSS appliqués correctement et code validé sans erreurs par le W3C. Bases solides pour la suite de la formation.

Structure sémantique : utilisation correcte de header, nav, main, section, article et footer.
Hiérarchie titres : h1 unique pour le titre principal, h2-h3 pour les sous-sections, logique respectée.
Styles CSS : couleurs harmonieuses, typographie lisible, espacements cohérents sur toute la page.
Images accessibles : tous les attributs alt renseignés avec descriptions pertinentes.
Validation W3C : zéro erreur HTML, zéro erreur CSS, code propre et bien indenté.
Fondamentaux acquis : compréhension du DOM, relation HTML/CSS, bonnes pratiques de développement.
