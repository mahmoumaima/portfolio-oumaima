Résumé du projet

Premier stage en agence web chez Webonzaï : le CTO Romain confie par email la mission d'actualiser la page de Riding Cities, une association de skate en région. Travail en autonomie pour mettre en pratique les bases HTML/CSS fraîchement acquises.

La mission consiste à créer une nouvelle section "Les membres fondateurs" avec noms et photos des créateurs, puis modifier la section de téléchargement pour proposer deux plannings distincts (enfants et adultes) au lieu d'un seul.

  • Section fondateurs : copier-adapter le code existant pour créer 3 colonnes avec textes, photos et attributs alt.
  • Boutons planning : ajouter un titre et dupliquer le bouton existant pour créer deux CTA avec couleurs/hover conformes à la maquette.
  • Réutilisation du code : s'appuyer sur les classes CSS déjà en place, pas de nouveau CSS à écrire.

Méthodologie

Approche d'intégration

Méthodologie pas-à-pas pour réussir l'intégration :

  1. Analyser le code existant : identifier les sections, balises et classes CSS
  2. Copier-coller : dupliquer la section "La mission de l'asso" pour créer "Membres fondateurs"
  3. Adapter le HTML : modifier textes, ajouter images avec attributs alt, ajuster colonnes
  4. Appliquer les classes CSS : utiliser les classes existantes pour obtenir le rendu de la maquette
  5. Valider : vérifier la conformité avec la maquette et tester les liens de téléchargement

Résultats finaux

Page Riding Cities mise à jour avec succès : nouvelle section "Les membres fondateurs" présentant les 3 créateurs de l'association avec photos et descriptions, et section "Découvrez nos cours" enrichie avec deux boutons de téléchargement distincts.

  • Réutilisation efficace : code HTML copié-adapté depuis les sections existantes, zéro ligne de CSS écrite.
  • Section fondateurs : 3 colonnes avec textes, photos et attributs alt renseignés pour l'accessibilité.
  • Boutons interactifs : deux CTA avec couleurs différentes, effets hover et liens vers les PDF de plannings.
  • Conformité maquette : respect pixel-perfect des espacements, couleurs et typographie Figma.
  • Pratique autonome : mission menée à bien sans supervision directe, premier livrable en stage.
