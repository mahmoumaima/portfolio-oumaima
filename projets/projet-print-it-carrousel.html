Résumé du projet

Premier projet freelance en JavaScript : Print it, une petite imprimerie familiale, souhaite dynamiser son site Internet statique. Le directeur Lucien envoie un email avec une vidéo explicative du carrousel idéal qu'il souhaite pour mettre en valeur leurs services.

Mission : transformer la bannière statique en carrousel interactif fonctionnel. Le site et le CSS sont déjà fournis, il faut uniquement coder la partie JavaScript pour gérer la navigation, les transitions et la synchronisation des éléments.

  • Étape 1 - Flèches statiques : ajouter les images des flèches (arrow_left.png, arrow_right.png) dans la bannière en suivant les indications CSS.
  • Étape 2 - Event listeners : mettre un listener sur chaque flèche, tester avec console.log/alert, différencier clic gauche vs clic droit.
  • Étape 3 - Bullet points : ajouter autant de points que d'images dans le tableau slides[], différencier le point actif avec classe CSS spécifique.
  • Étape 4 - Changement d'images : au clic flèche droite → changer bullet actif, image et texte (tagLine). Idem flèche gauche pour éléments précédents.
  • Étape 5 - Boucle infinie : conditions pour gérer dernière image → première (clic droit) et première → dernière (clic gauche).

Résultats finaux

Carrousel JavaScript entièrement fonctionnel pour Print it : navigation fluide par flèches, bullet points synchronisés avec l'image active, changement automatique des textes et boucle infinie sans accroc.

  • Event listeners : flèches gauche/droite détectent les clics, différenciation correcte des directions.
  • Bullet points dynamiques : génération automatique (1 par slide), classe CSS active sur le bullet courant.
  • Navigation complète : clic flèche droite → image suivante + texte + bullet actif. Idem pour flèche gauche.
  • Boucle infinie : dernière image → retour automatique à la première (et vice-versa), aucune erreur console.
  • Manipulation DOM : innerHTML pour textes, construction chemin images, gestion index tableau avec .length.
  • Code propre : debuggage avec console.log, conditions if/else claires, code JavaScript vanilla sans librairie.
