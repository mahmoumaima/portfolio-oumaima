Le Contexte

Comprendre le challenge

Mission

Premier projet freelance en JavaScript : Print it, une petite imprimerie familiale, souhaite dynamiser son site Internet statique. Le directeur Lucien envoie un email avec une vidéo explicative du carrousel idéal qu'il souhaite.

Objectif

Transformer la bannière statique en carrousel interactif fonctionnel. Le site et le CSS sont déjà fournis, il faut uniquement coder la partie JavaScript pour gérer la navigation, les transitions et la synchronisation.

Livrables

  • Flèches de navigation
  • Bullet points synchronisés
  • Boucle infinie
  • Code JavaScript propre

Les Résultats

Mission accomplie

Carrousel JavaScript entièrement fonctionnel pour Print it : navigation fluide par flèches, bullet points synchronisés avec l'image active, changement automatique des textes et boucle infinie sans accroc.

Event listeners : flèches gauche/droite détectent les clics, différenciation correcte des directions.
Bullet points dynamiques : génération automatique (1 par slide), classe CSS active sur le bullet courant.
Navigation complète : clic flèche droite → image suivante + texte + bullet actif. Idem pour gauche.
Boucle infinie : dernière image → retour automatique à la première (et vice-versa), aucune erreur console.
Manipulation DOM : innerHTML pour textes, construction du chemin des images, gestion de l'index avec .length.
Code propre : debugging avec console.log, conditions if/else claires, JavaScript vanilla sans librairie.
