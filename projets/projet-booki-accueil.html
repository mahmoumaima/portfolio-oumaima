Le Contexte

Comprendre le challenge

Mission

Début d'alternance chez Booki, start-up qui développe un site pour trouver hébergements et activités. Sarah (CTO) et Loïc (UI designer) fournissent la maquette Figma, une note de synthèse et une base de code (index.html, css, images) à complèter.

Objectif

Intégrer l'interface en HTML/CSS en respectant les spécifications (structure de dossiers, polices, liens et classes existants), puis adapter la page aux formats tablette et mobile avec des media queries.

Livrables

  • Header et navigation intégrés
  • Recherche, filtres et cartes d'hébergements
  • Section activités et footer
  • Responsive tablette/mobile
  • Validation W3C HTML/CSS

Détails clés

  • Client Booki, start-up hébergements et activités
  • Équipe Sarah (CTO) et Loïc (UI designer)
  • Rôle Développeur Web (Alternance)

Stack & Outils

HTML5 CSS3 Flexbox Media Queries Figma DevTools & W3C

Les Résultats

Mission accomplie

Interface Booki intégrée à partir de la maquette et de la base de code : recherche, filtres, hébergements, populaires, activités et footer, puis adaptation responsive et validation W3C.

Mise en page Flexbox maîtrisée : navigation, filtres, cartes et footer organisés avec direction adaptative.
Cartes hébergements intégrées : 6 cartes développées avec images object-fit, ombres CSS et attributs alt.
Formulaire de recherche créé : bouton texte desktop / icône loupe mobile avec styles cohérents.
Section activités réalisée : 4 cartes en colonnes intégrées avec hauteur identique.
Design responsive implémenté : media queries tablette/mobile avec ordre d'affichage adaptatif.
Code validé W3C : HTML/CSS conformes aux standards, structure sémantique respectée.
