Le Contexte

Comprendre le challenge

Mission

Renfort front-end pour l'agence ArchiWebos (50 salariés). Charlotte, cheffe de projet, confie la finalisation du site portfolio de Sophie Bluel, architecte d'intérieur. Un designer et un développeur back ont déjà travaillé sur le projet.

Objectif

Développer la page de présentation des travaux à partir du HTML fourni, créer la page de connexion administrateur, et coder la modale permettant d'uploader de nouveaux médias. Le back-end fonctionne via une API REST.

Livrables

  • Galerie dynamique filtrée
  • Page connexion JWT
  • Modale édition travaux
  • Upload avec FormData

Détails clés

  • Client Sophie Bluel, architecte d'intérieur
  • Agence ArchiWebos (50 salariés)
  • Rôle Développeur Front-End
  • Durée 70h

Stack & Outils

JavaScript HTML5 CSS3 Fetch API Node.js Swagger

Les Résultats

Mission accomplie

Portfolio architecte entièrement fonctionnel : galerie dynamique récupérée depuis l'API, filtrage par catégories, authentification JWT sécurisée et interface d'administration complète avec upload/suppression de travaux.

Galerie dynamique : fetch API GET /api/works, affichage des travaux avec images/titres, filtres boutons actifs.
Authentification : page login fonctionnelle, POST /api/users/login, token JWT stocké localStorage.
Mode édition : bandeau noir "Mode édition", liens "modifier" sur sections, logout pour quitter.
Modale suppression : affichage travaux en miniature, icône poubelle, DELETE /api/works/{id} avec token.
Upload photos : formulaire input file + titre + catégorie, preview image temps réel, POST FormData.
Mise à jour live : suppression/ajout rafraîchit galerie page ET modale sans recharger la page.
