Le Contexte

Comprendre le challenge

Mission

Projet final de la formation : créer un portfolio professionnel pour présenter les compétences et projets de testeur logiciel. Le portfolio doit mettre en valeur les 11 projets réalisés pendant la formation.

Objectif

Concevoir un portfolio responsive avec HTML/CSS, présenter clairement les projets avec détails techniques, et déployer sur GitHub Pages. Le portfolio sert de vitrine professionnelle pour les candidatures.

Livrables

  • Portfolio en ligne
  • Repository GitHub
  • Pages projets détaillées
  • Optimisations SEO

Détails clés

  • Formation OpenClassrooms Testeur Logiciel
  • Objectif Vitrine professionnelle pour recruteurs
  • Rôle Développeur Front-End & QA
  • Durée 80h

Stack & Outils

HTML5 CSS3 JavaScript GitHub Pages Lighthouse Git

Les Résultats

Mission accomplie

Portfolio professionnel testeur logiciel déployé sur GitHub Pages : 12 projets présentés, optimisations performance/SEO, design responsive et accessible en ligne pour les recruteurs.

Structure complète : page accueil avec présentation, section compétences QA, galerie des 12 projets formation.
Pages projets détaillées : chaque projet avec contexte, stack technique, défis et captures d'écran.
Design responsive : adaptation mobile/tablette/desktop avec breakpoints CSS, navigation hamburger mobile.
Optimisations performance : images WebP compressées, lazy loading, minification, score Lighthouse 95+.
SEO optimisé : balises meta (title, description), Open Graph pour réseaux sociaux, sitemap.xml.
GitHub Pages : déploiement réussi, URL publique active, README documentation complète.
