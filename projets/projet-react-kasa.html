Le Contexte

Comprendre le challenge

Mission

Kasa, leader français de la location d'appartements entre particuliers avec plus de 500 annonces, entreprend une refonte complète de sa plateforme web vieillissante (ASP.NET, 15 ans) vers une architecture moderne JavaScript.

Objectif

Développer l'intégralité du front-end React de la nouvelle plateforme, en respectant les maquettes Figma et les coding guidelines fournies. Le backend (NodeJS) étant en cours de développement, utilisation d'un fichier JSON simulant les données de 20 logements.

Livrables

  • Application React complète
  • Routing SPA multi-pages
  • Composants réutilisables
  • Design responsive

Détails clés

  • Client Kasa, leader location appartements France (500+ annonces)
  • CTO Laura
  • Rôle Développeur Front-End Freelance
  • Durée 60h

Les Résultats

Mission accomplie

Application Kasa complète en React : navigation fluide avec React Router, 20 logements affichés en galerie, pages détaillées avec carrousel photos et composants interactifs, responsive sur tous formats.

React Router SPA : 4 routes configurées (/, /about, /logement/:id, /404), navigation sans rechargement.
Page Accueil : galerie 20 cards responsive, clic card → navigation vers page logement correspondante.
Page Logement : route dynamique avec ID, affichage toutes données (titre, localisation, tags, host, rating 5 étoiles, carrousel photos).
Carrousel photos : flèches navigation (masquées si 1 seule photo), numérotation "1/4", changement image au clic.
Composants Collapse : description et équipements dépliables, animation chevron 180°, fermés par défaut.
Page 404 : route inexistante ou ID invalide → redirection automatique vers page erreur 404.
Composants réutilisables : Card, Carousel, Collapse, Rating, Tag codés en modules indépendants.
Responsive complet : maquettes desktop/mobile respectées, breakpoint tablette défini logiquement.
Code quality : Sass organisé, un composant par fichier, aucun warning console, guidelines respectées.
