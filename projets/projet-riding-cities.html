Le Contexte

Comprendre le challenge

Mission

Premier stage en agence web chez Webonzaï : le CTO Romain confie par email la mission d'actualiser la page de Riding Cities, une association de skate en région. Travail en autonomie pour mettre en pratique les bases HTML/CSS.

Objectif

Créer une nouvelle section "Les membres fondateurs" avec noms et photos des créateurs, puis modifier la section de téléchargement pour proposer deux plannings distincts (enfants et adultes).

Livrables

  • Section fondateurs
  • Boutons téléchargement
  • Code conforme maquette
  • Attributs alt renseignés

Détails clés

  • Client Riding Cities, association de skate
  • Agence Webonzaï
  • Rôle Développeur Junior (Stagiaire)

Stack & Outils

HTML5 CSS3 Figma Accessibilité GitHub

Les Résultats

Mission accomplie

Page Riding Cities mise à jour avec succès : nouvelle section "Les membres fondateurs" présentant les 3 créateurs et section "Découvrez nos cours" enrichie avec deux boutons de téléchargement distincts.

Réutilisation efficace : code HTML copié-adapté depuis les sections existantes, zéro ligne de CSS écrite.
Section fondateurs : 3 colonnes avec textes, photos et attributs alt renseignés pour l'accessibilité.
Boutons interactifs : deux CTA avec couleurs différentes, effets hover et liens vers les PDF des plannings.
Conformité maquette : respect pixel-perfect des espacements, couleurs et typographie Figma.
Accessibilité : tous les attributs alt renseignés, structure sémantique respectée.
Pratique autonome : mission menée à bien sans supervision directe, premier livrable en stage.
