Le Contexte

Comprendre le challenge

Mission

Mission consultante test chez PayForge, entreprise développant des solutions bancaires pour clients internationaux. Amandine, Product Owner de l'équipe Tomsen, confie l'élaboration de la stratégie de test pour nouvelle fonctionnalité virements internationaux.

Objectif

Analyser les exigences critiques, concevoir une stratégie de test complète, créer le cahier de recette avec cas de test détaillés, planifier les tests par sprint et organiser la campagne de non-régression finale.

Livrables

  • Stratégie de test
  • Cahier de recette
  • Matrice des risques
  • Planning tests

Détails clés

  • Client PayForge, solutions bancaires internationales
  • Product Owner Amandine (équipe Tomsen)
  • Rôle Consultante Test / QA Lead
  • Durée 40h (3 sprints x 2 semaines)

Les Résultats

Mission accomplie

Stratégie de test complète pour Tomsen PayForge : revue des exigences, analyse des risques, cahier de recette détaillé et planification des tests sur 3 sprints avec jalons qualité.

Revue exigences : analyse document Amandine, 90% exigences testables, 5% non testables, questions clarification formulées.
Stratégie de test : objectifs qualité définis (sécurité transactions, fiabilité virements), périmètre précisé.
Analyse risques : matrice probabilité/impact avec 15 risques, priorisation validation montant, conversion devises, sécurité.
Cahier de recette : 45 cas de test détaillés format standardisé (ID, user story, préconditions, étapes, résultats).
Planning tests : répartition sur 3 sprints (15 + 18 + 12 cas), estimation 120h avec abaque.
Jalons qualité : critères GO/NO GO par sprint (80% → 85% → 95% + zéro bloquant), 20 tests non-régression.
