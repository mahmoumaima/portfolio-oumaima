Le Contexte

Comprendre le challenge

Mission

Début d'alternance chez Booki, start-up de réservation d'hébergements et d'activités. La CTO Sarah lance la refonte du site avec une nouvelle interface conçue par Loïc, l'UI designer.

Objectif

Intégrer la page d'accueil en HTML5 et CSS3 avec un design entièrement responsive. Maquette desktop fournie, adaptation tablette/mobile à réaliser avec Media Queries.

Livrables

  • Page d'accueil responsive
  • Code HTML sémantique
  • CSS Flexbox optimisé
  • Validation W3C

Détails clés

  • Client Booki, plateforme de réservation
  • Équipe Sarah (CTO) & Loïc (UI Designer)
  • Rôle Développeur Web (Alternance)
  • Durée 40h

Stack & Outils

HTML5 CSS3 Flexbox Media Queries Figma GitHub

Les Résultats

Mission accomplie

Page d'accueil Booki entièrement responsive : adaptation fluide desktop/tablette/mobile avec Flexbox et Media Queries. Interface fidèle à la maquette Figma, code sémantique validé W3C et performances optimisées.

Responsive design : 3 breakpoints gérés (desktop 1024-1440px, tablette ≤1024px, mobile <768px).
Flexbox maîtrisé : navigation, filtres, cartes et footer organisés avec Flexbox, direction adaptative.
Cartes hébergements : grille 3 colonnes → 1 colonne mobile, images object-fit, ombres CSS subtiles.
Formulaire recherche : bouton texte desktop / icône loupe mobile, styles cohérents.
Accessibilité : tous les attributs alt renseignés, navigation clavier, contrastes respectés.
Code propre : HTML sémantique (header/nav/main/section/footer), CSS organisé, validation W3C 100%.
Performance : images optimisées, CSS épuré avec pixels pour margins/paddings et % pour widths.
